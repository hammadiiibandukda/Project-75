# Project-75

Journey to losing weight — a personal weight & workout tracker.

87 kg → 70 kg, weekly weigh-ins. Built with React + Vite, persists to `localStorage`.

## Features

- Log weekly weigh-ins, see trend over the last 12 entries on an SVG chart
- Track lost / remaining kg, week-over-week change, and overall progress %
- Log workout sessions (multiple exercises per session, minutes per exercise)
- All data stored locally in your browser

## Run locally

```sh
npm install
npm run dev
```

Then open the URL Vite prints (default `http://localhost:5173`).

## Build

```sh
npm run build
npm run preview
```
